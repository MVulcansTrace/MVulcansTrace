## Featured Project

### [VulcansTrace-Windows](https://github.com/MVulcansTrace/VulcansTrace-Windows)

Windows desktop firewall-log threat detection portfolio built with C#/.NET and WPF.

What it demonstrates:

- Windows Firewall log parsing with validation, raw-line preservation, and parse-error reporting
- Behavioral detectors for port scans, beaconing, lateral movement, flood behavior, policy violations, and novelty
- MITRE ATT&CK-aligned documentation for detection logic and analyst interpretation
- Cross-signal risk escalation when multiple findings align on the same source host
- HMAC-backed evidence exports for post-export package integrity verification
- WPF desktop UI with asynchronous analysis, filtering, cancellation, and export workflows
- 223 passing automated tests across parser, detector, evidence, integration, and UI paths
